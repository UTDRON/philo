# Gamified Learning App for Philosophy
# Playstore Review Scraper

This repo is a part of the IE535 HCI project at University at Buffalo(Fall 2025).

For gathering user requirements data for designing a "Gamified Learning App for Philosophy" we scraped user reviews for famous gamified learning apps- Duolingo, Kahoot, Khan Academy and Quizizz- from play store.

The repo contains both scraping scripts as well as scraped review data.
